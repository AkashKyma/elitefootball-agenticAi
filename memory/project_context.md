# Project Context

## Project Goal
Build the foundation for `elitefootball-agenticAi`, a Python-based multi-agent system focused on collecting, organizing, and analyzing data about IDV players.

## MVP Scope
The MVP is intentionally narrow:
- focus on IDV players only
- collect core player profile data
- support scraping, storage, and future analysis workflows
- keep the architecture small enough to evolve safely

## Working Rules
All future tasks MUST:
- read memory before work
- update memory after work
